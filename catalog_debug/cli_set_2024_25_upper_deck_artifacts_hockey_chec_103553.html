2024-25 Upper Deck Artifacts Hockey Checklist Rundown

Rookie redemptions take center focus in the 2024-25 Upper Deck Artifacts Hockey checklist. These add to the main series and offer collectors an early chance for top names to make their official debut during the '24-25 season.

Artifacts NHL also boasts a few on-card autograph sets and the next round of the UD Bounty program.

Base & Parallel Details

The base set for 2024-25 Upper Deck Artifacts Hockey is filled with talent via a few different groupings. The first 200 cards cover Veterans (#1-100), Goalies (/699; #101-120), Stars (/699; #121-155), Legends (/599; #156-170) and Rookies (/999; #171-200).

Each tier has multiple parallels with varying print runs, such as Foiboard (1:20 packs), Gold Foilboard (1:40 packs), Plexi (1:60 packs) and Masterful Artifacts (1/1). Look for four serial-numbered base cards or parallels per Hobby box. Seafoam Green (/10) is a new addition, while Neon Green is only inserted in Blasters.

There are also limited autographs, relics and auto relics from the base player lineup.

Rookie Redemption Details

The final 50 cards in the 2024-25 Upper Deck Artifacts Hockey base set are saved for Rookie Redemptions (#201-250). Select names have cards in the Horizontal SP Photo Variants set.

These redemptions combine with acetate inserts to fall once per Hobby box.

Key names appear in Rookie Redemptions Auto (I-VI, VII-XII and XIII-XVIII), Auto Horizontal Variation (SP1-SP5) and Rookie Relic Redemptions (I-VI).

Insert Details

As for the acetate cards in 2024-25 Upper Deck Artifacts Hockey, the product works in the all-new Antiquity (1:100 packs) and Antiquity Rookies (1:200 packs).

There are also Clear Cut Artifacts Rookies (1:67 packs) and Clear Cut Artifacts Rookie Update (1:55 packs) cards.

Autograph & Relic Details

Shifting to the autographs and relics, 2024-25 Upper Deck Artifacts Hockey supplies old favorites alongside debuts.

Auto Facts (1:50 packs) is back for another run in Artifacts NHL.

Fans may also remember NHL Remnants (1:50 packs), featuring game-worn gear. Per UD, "Try to collect all the cards in this set to complete the hidden puzzle."

The unique Fresco Signatures (1:150 packs) set provides an artistic flair to the release.

More new hits arrive in Orchestrators (/25), Stanley Cup Showdown Jerseys (1:200 packs) and Patches (/25).

In addition, several on-card autograph sets pick up from last year. Dubbed "living sets" by UD, the hard-signed continuations include Admirable Impressions Signatures (/24), Esteemed Endorsements (/28) and Signature Apparel (/36 or /12). These are exclusive to Hobby boxes.

Memorabilia takes various forms in 2024-25 Upper Deck Artifacts Hockey. Arena Artifacts (/67) is dedicated to the Philadelphia Flyers' Spectrum Arena (1967-1996).

Inaugural Access Ticket is a Hobby-exclusive series with Modern (1980-Present) and Vintage (1967-1979) editions. These showcase authentic ticket stubs for debut seasons across various NHL franchises.

2024-25 Upper Deck Artifacts Hockey UD Bounty Details

Finally, 2024-25 Upper Deck Artifacts Hockey provides an added chase via the Aurum inserts and the UD Bounty Program.

The goal is to collect the full 48-card set and everyone that redeems this set can earn the final two bonus cards (#49 and #50). If you are among the first 25 to redeem, more prizes await.

  • The first 3 collectors get the 12-card Aurum set, plus one Aurum Autograph and one Aurum Dual Autograph.
  • The next 7 collectors get the 12-card Aurum set and one Aurum Autograph.
  • The next 15 collectors get the 12-card Aurum set.

However, Aurum is not just for UD Bounty as there are pack-pulled Aurum Signatures cards numbered to 25 copies.

2024-25 Parkhurst Champions Hockey is Upper Deck's throwback-inspired NHL product known for old-school designs and vintage callbacks.

One change this year for the Hobby format is that the box hit can be an autograph, relic or lenticular insert. There is also a new Retro Bonus pack in every Hobby box.

2024-25 Parkhurst Champions Hockey Checklist Rundown

The 2024-25 Parkhurst Champions Hockey checklist highlights the past, present and future of the league in the distinct Parkhurst style.

2024-25 Parkhurst Champions Hockey Base & Parallel Details

Offering 300 cards, 2024-25 Parkhurst Champions Hockey features subsets for Base (#1-200; 6 per pack), Base Stars SPs (#201-240; 1:8 packs), Base Rookies SPs (#241-280; 1:8 packs) and Base Greats SPs (#281-300; 1:17 packs) cards.

In total, each Hobby box should deliver six Base SPs along with 20 Silver parallels, six Rainbow Foil parallels and one serial-numbered card.

2024-25 Parkhurst Champions Hockey Base Parallels List

  • Blank Back
  • Silver - 1 per pack
  • Rainbow Foilboard - 6 per box
  • Shell Pink Border - /99
  • Baby Blue Border - /25
  • Black Border - /10

2024-25 Parkhurst Champions Hockey Insert Details

On average, another six cards in each Hobby box are pulled from the insert selection. 2024-25 Parkhurst Champions Hockey displays 1950s-styled art as the main background of Atom Squad (1:33 packs).

More options arrive with Crease Lightning (1:11 packs), Saturday Night Standouts (1:14 packs) and
Shake, Rattle, and Roll (1:17 packs).

Trivia buffs should enjoy You Wouldn’t Believe! (1:23 packs), which covers some of the top names of the past.

A possible box hit, the short-printed lenticular inserts include In Full Color (1:216 packs) and Retro-Active (1:270 packs).

2024-25 Parkhurst Champions Hockey Autograph & Relic Details

As for the autographs and relics, most of the hits in 2024-25 Parkhurst Champions Hockey work from the base lineup.

Uncover signatures via Base Auto (1:80 packs), Base Stars Auto (1:180 packs), Base Rookies Auto (1:72 packs) and Base Greats Auto (1:1,200 packs). Limited Blue versions are capped at 49 copies or less.

Memorabilia choices appear in Base Stars Jersey (1:360 packs), Base Rookies Jersey (1:120 packs) and Base Greats Jersey (1:720 packs), as well as Base Stars Patch Auto (/15), Base Rookies Patch Auto (/25) and Base Greats Patch Auto (/10).

The Base Stars subset also has a limited Patch (/25) version and all three subsets (Stars, Rookies and Greats) offer the one-of-one Gold Patch Auto edition.

On top of that, Parkhurst Cuts adds a one-of-one chase with Championship Salute '53-54 Red Wings, Class of 1953-54 and Stars of the '50s.

2024-25 Parkhurst Champions Hockey Retro Bonus Pack Details

A new addition to Hobby boxes is the 2024-25 Parkhurst Champions Hockey Retro Bonus pack. This five-card pack is the only place to find the 1953-54 Retros inserts, displaying a more modern take on the classic design.

Limited versions include Jumbo Gum Back (1 per pack), Rainbow Foilboard (/53), Blank Back (/15), Black Rainbow Foilboard (1/1) and Auto (#'d).

  2. Hockey Cards

Release Date: April 23, 2025

Catering to memorabilia collectors, 2024-25 SP Game Used Hockey highlights past seasons with a mix of event-used and game-used NHL gear.

Although each Hobby box still has four hits (auto, auto relic, relic or tech card), the box size increases to nine cards. Last year, these Upper Deck boxes contained six cards.

2024-25 SP Game Used Hockey Checklist Rundown

The 2024-25 SP Game Used Hockey checklist has grown to accommodate the additional cards per box with five base/inserts to go with the four hits.

2024-25 SP Game Used Hockey Base & Authentic Rookies Details

2024-25 SP Game Used Hockey opens with 200 veterans and legends in the base set, falling once per box.

2024-25 SP Game Used Hockey Base Parallels List

  • Red - 1:2.2
  • Gold - /149
  • Rainbow Foil - jersey #'d
  • Purple - /5 (some signed)
  • Green - 1/1 (some signed)

The Authentic Rookies subset (#201-280; 1:1.25) adds 80 more cards to the main lineup.

2024-25 SP Game Used Hockey Authentic Rookies Parallels List

  • Red - 1:2.5
  • Crystal
  • Gold - /149
  • Rainbow Foil - jersey #'d
  • Purple - /5 (some signed)
  • Green - 1/1 (some signed)

Select names are treated to autographs across the base and Authentic Rookies sets.

Limited relics and autograph relics are also available.

2024-25 SP Game Used Hockey Insert Details

As for the inserts, New Grooves (1:5) combines with the base and Authentic Rookies cards to average five per box, including three standard cards and two parallels.

In addition, 2024-25 SP Game Used Hockey adds the new Visible Spectrum (1:12) series. Per Upper Deck, the cards showcase "a design inspired by waves of light."

2024-25 SP Game Used Hockey Autograph & Relic Details

The rest of the 2024-25 SP Game Used Hockey checklist is saved for the hits.

Displaying a hard-signed manufactured letter patch, fans can again spell out the last name of top rookies via Draft Day Marks (/35).

HOF Marks is a new manufactured relic series for former greats, but it's not signed.

Other favorites on the relic side include Inked Rookie Sweaters (/99), Rookie Jumbo Patch (/25), Supreme Materials (/15) and Supreme Patches (/15).

A new option for goalies appears in Websites Glove Relics (/15). These contain game-used glove pieces.

On top of that, 2024-25 SP Game Used Hockey revisits past NHL events with memorabilia, such as the All-Star Game, Global Series, Heritage Classic, Stanley Cup and Winter Classic.

This includes event-used and game-used swatches in Banner Year, Fabrics, Net Cord, Puck Relics and more.
